\section{Custom Commands Demo}

\subsection{Boxes}
We have built-in boxes for important information:

\begin{infobox}[Did you know?]
LaTeX handles numbering automatically. You never have to manually update "Figure 1.1" ever again.
\end{infobox}

\begin{warningbox}
We are now using \textbf{LuaLaTeX} to compile, which supports modern Variable Fonts better than XeLaTeX.
\end{warningbox}

\subsection{Mixed Language Support}
If you enabled Arabic in the config, you can write:
\begin{center}
    \textarabic{السلام عليكم ورحمة الله وبركاته}
\end{center}
Or English text: \keyword{Hello World}.
