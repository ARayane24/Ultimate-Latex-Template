\section{Introduction}
This template is designed to be a robust alternative to MS Word. It keeps content separate from styling.

\subsection{How to Configure}
Go to \texttt{config/settings.tex} to change:
\begin{itemize}
    \item The document language (Arabic/English).
    \item The direction (RTL/LTR).
    \item The theme color.
    \item Fonts and Margins.
\end{itemize}
